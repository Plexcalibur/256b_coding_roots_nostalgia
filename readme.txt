Nostalgia - our first demo after a 22 year hiatus and also our first attempt on sizecoding an intro
Release: Evoke 2023
Compo: 256 Byte Intro
Platform: MS-DOS
Size: 256 bytes

Effects: RotoZoom, Plasma, Motion Blur, Fire, Custom Palette
Coder/Group: Plex/BionFX
Artist/Group: Imagician/BionFX

Build: `nasm -f bin -o bfx_256a.com bfx_256a.asm`
Run: in Dosbox ( https://www.dosbox.com/ ) with the supplied config
